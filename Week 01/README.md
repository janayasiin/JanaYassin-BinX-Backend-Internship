# Week 01 - Onboarding, C# Fundamentals & Git


## 🚀 Daily Curriculum & Topics Overview

* **Day 01: Program Orientation & .NET Development Environment Setup**
  * Installed and verified the .NET SDK.
  * Navigated the `dotnet CLI` (`dotnet new`, `dotnet build`, `dotnet run`) to manage console applications.
  * Configured and tested the local development environment and IDE workspace.

* **Day 02: C# Fundamentals I (Types, Variables & Control Flow)**
  * Explored value types vs. reference types and their memory behaviors.
  * Applied type inference (`var`) and proper naming conventions.
  * Built robust control flow using conditional statements, pattern-matching switch expressions, and loops.
  * Handled safety with Nullable Reference Types.

* **Day 03: C# Fundamentals II (Object-Oriented Programming)**
  * Modeled domain entities utilizing `Classes`, immutable `Records`, and `Structs`.
  * Applied encapsulation using access modifiers and private backing fields.
  * Implemented inheritance, contracts via `Interfaces`, and polymorphism in practice.

* **Day 04: C# Fundamentals III (Collections & LINQ Basics)**
  * Selected optimal collection types (`List<T>`, `Dictionary<TKey, TValue>`, `HashSet<T>`).
  * Queried and transformed data using LINQ (Method syntax and Query syntax).
  * Implemented basic asynchronous programming concepts (`async/await`) and robust exception handling.

* **Day 05: Git & GitHub Workflow & Synthesis**
  * Mastered core Git commands and feature-branch workflows.
  * Practiced writing descriptive commit messages and managing clean repository histories.
  * Synthesized the week's milestones into a structured pull request.

## 🛠️ Technical Stack
* **Language & Runtime:** C#, .NET SDK (LTS)
* **Development Environment:** Visual Studio Code / Visual Studio
* **Version Control:** Git & GitHub
